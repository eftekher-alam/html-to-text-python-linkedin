1. Install python 3.11.1
2. Install beautifulsoup
	command : pip install beautifulsoup4 --user
3. install  requests
	command : pip install requests --user
	
4. run the app.py 
	command : python app.py
	

Input note: past the input HTML in the body tag of index.html file.

Output Note : You will get the output as plane text in the file.txt file.
